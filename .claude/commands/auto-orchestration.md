# Full-Stack Team Auto-Orchestration

## Orchestration Objective
Coordinate full-stack team members for comprehensive application development from architecture to deployment.

## Workflow Sequence

### Phase 1: Architecture & Design (Software Architect â†’ UI/UX Designer)
1. **Software Architect** - Define system architecture, technology stack, design patterns, and technical specifications
2. **UI/UX Designer** - Create user experience flows, wireframes, prototypes, and design system components

### Phase 2: Database Foundation (Database Administrator)
3. **Database Administrator** - Design database schema, optimize queries, set up data relationships, and implement data security

### Phase 3: Full-Stack Development (Full-Stack Developer)
4. **Full-Stack Developer** - Implement both frontend and backend components, integrate APIs, and build end-to-end features

### Phase 4: Security & Compliance (Security Engineer)
5. **Security Engineer** - Conduct security audits, implement security controls, perform vulnerability assessments, and ensure compliance

### Phase 5: Performance Optimization (Performance Engineer)
6. **Performance Engineer** - Analyze performance bottlenecks, implement optimization strategies, conduct load testing, and monitor system performance

## Coordination Rules
- **Architecture First**: All development follows architectural decisions and design guidelines
- **Database-Driven**: Database design influences application structure and data flow
- **User-Centered**: UI/UX designs guide frontend implementation decisions
- **Security by Design**: Security considerations integrated throughout all phases
- **Performance-Driven**: Performance optimization integrated throughout the development lifecycle

## Parallel Work Opportunities
- UI/UX Designer and Database Administrator can work simultaneously after architecture is defined
- Security Engineer can review designs and provide security requirements early in the process
- Performance Engineer can continuously monitor and optimize performance throughout development
